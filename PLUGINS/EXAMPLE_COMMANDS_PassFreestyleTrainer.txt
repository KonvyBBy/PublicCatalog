# ===================================
# BASIC SETUP
# ===================================

# Enable the plugin

# Set training mode (choose one)

# pft mode aerial_control    # For air control
# pft mode flip_reset        # For flip resets
# pft mode air_dribble       # For air dribbles
# pft mode ceiling_shot      # For ceiling shots

# pft difficulty intermediate  # Medium, 1.5x points
# pft difficulty advanced      # Hard, 2.0x points

# ===================================
# TROUBLESHOOTING
# ===================================

# Plugin not working?
# 1. Check if enabled: pft enable
# 2. Verify mode is set: pft mode pass_trainer
# 3. Check difficulty: pft difficulty beginner
# 4. Look for error messages in console

# Not getting points?
# 1. Ensure you're meeting mode requirements
# 2. Check pass speed (must be 500+)
# 3. Verify you're airborne for aerial modes
# 4. Make sure plugin is enabled

# Commands not recognized?
# 1. Check spelling (case doesn't matter)
# 2. Ensure plugin loaded successfully
# 3. Look for startup message in console
# 4. Restart bot if needed
